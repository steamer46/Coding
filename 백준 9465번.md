## 아이디어
일단 최대값을 구하는거기 때문에 단순계산으로는 풀기 어려울것같습니다. 하지만, 규칙이 명확하게 존재합니다. "같은 변을 공유하지 않는다"를 이용하여 점화식을 만들어낼 수 있을거 같아서 dp를 고르게 되었습니다





### 하지만
메인이 되는 점화식을 먼저 만들어 보겠습니다
각 위치에서 최대값을 구하기 위해서는 이전의 상황에 어떻게 끝났는지를 알아야합니다. 


이전 공주님을 구하라 문제에서도 같은 위치라고 하더라도 상황이다르면 차원하나를 추가한것 처럼, 
여기서도 같은 위치더라도 상황을 다르게 표현할 수 있는 차원을 추가해서 만들어 봅시다



dp[순서][현재 위치에서 상태] 이런 식으로 만들면 아래와 같이 만들 수 있습니다 
// 위dp[0][0] dp[1][0] ..... 
// 아래 dp[0][1] .....
// 아무것도 x dp[0][2].....




따라서 점화식은 다음과 같이 만들 수 있습니다
<img width="822" height="209" alt="image" src="https://github.com/user-attachments/assets/297b3137-4704-46d9-885c-145f31d6e0d6" />


이후에 i를 1부터 시작하기 때문에 arr[0][0]부분은 직접 초기화를 해줘야합니다
따라서
dp[0][0] =0;
dp[0][1] =arr[0][0];
dp[0][2] =arr[1][0];

​

처음은 이런식으로 초기화를 해줘야합니다



​​이 코드들을 모두 합쳐서 살을 붙힌 코드는 이렇게 됩니다
//결과물

```cpp
#include <iostream>

#include <algorithm>

using namespace std;

​

int T,N;

int arr[2][100001];

int dp[100001][3];

​

int main() {

ios:: sync_with_stdio(false);

cin.tie(NULL);

​

cin>> T;

​

while(T--){

cin>>  N;

​

for(int i=0; i<2; i++){

      for(int j=0; j<N; j++){

              cin >> arr[i][j];

          }

      }
dp[0][0] = arr[0][0];

dp[0][1] = arr[1][0];

dp[0][2] = 0;

​

// 위dp[0][0] dp[1][0] ..... 

// 아래 dp[0][1] .....

// 아무것도 x dp[0][2].....

​

for(int i=1;i < N ;i++) {

dp[i][0] = max(dp[i-1][1], dp[i-1][2])+arr[0][i];

dp[i][1] = max(dp[i-1][0], dp[i-1][2])+ arr[1][i];

dp[i][2] = max({dp[i-1][0],dp[i-1][1], dp[i-1][2]});

}

​

cout<< max({dp[N-1][0], dp[N-1][1], dp[N-1][2]})<< '\n';

}// while문 끝

return 0;

}// 메인함수 끝

​

```

## 실수했던 부분
실수 한것은 딱히 없이 바로 맞출수있었습니다

​다만! 알아둬야 할것은 
​

#### * max를 쓸때 #include<algorithm>을 필수로 적어야 합니다

따라서  max({dp[i-1][0],dp[i-1][1], dp[i-1][2]}); 이런식으로 초기화 해줘야 합니다.

이후에 메인 함수에서 가상의 0번 정점에서 모든 정점으로 가는 가선추가 합니다. 그 이유는 모든 간선끼리 연결이 된것이 아니고,  사이클 자체가 끊어진 곳이 존재할 수 도 있기때문에, 그것을 방지하기 위해서 가상의 정점에 연결을 시켜줍니다

// 0의 정점연결

이후에 코드를 완성시켜 주면 다음과 같습니다


```cpp

#include <iostream>

#include <vector>

#include <tuple>

#include <algorithm>

using namespace std;

​

const long long INF = 1e18;

int TC, N, M, W;

vector<tuple<int, int, int>> edges;

long long dist[501];

​

bool bellmanFord() {

    fill(dist, dist + N + 1, INF);

    dist[0] = 0; // 가상의 시작점

    for (int i = 0; i < N; i++) {

        for (auto [u, v, cost] : edges) {

            if (dist[u] != INF && dist[u] + cost < dist[v]) {

                dist[v] = dist[u] + cost;

            }

        }

    }

    for (auto [u, v, cost] : edges) {

        if (dist[u] != INF && dist[u] + cost < dist[v]) {

            return true; // 음의 사이클 존재

        }

    }

    return false;

}

​

int main() {

    ios::sync_with_stdio(false);

    cin.tie(0);

    cin >> TC;

    while (TC--) {

        cin >> N >> M >> W;

        edges.clear();

        for (int i = 0; i < M; i++) {

            int a, b, c;

            cin >> a >> b >> c;

            edges.push_back({a, b, c});

            edges.push_back({b, a, c}); // 양방향 도로

        }

        for (int i = 0; i < W; i++) {

            int a, b, c;

            cin >> a >> b >> c;

            edges.push_back({a, b, -c}); // 웜홀: 음의 간선

        }

        // 가상의 0번 정점에서 모든 정점으로 가는 간선 추가

        for (int i = 1; i <= N; i++) {

            edges.push_back({0, i, 0});

        }

        if (bellmanFord()) {

            cout << "YES\n";

        } else {

            cout << "NO\n";

        }

    }

    return 0;

}

```


## 알아두어야 할것

웜홀이 아닌 일반적인 정점에서 정점을 연결하는 경우에는 꼭 양방향 간선으로 edges에 저장을 해야합니다
단방향 간선으로 하는 경우에는 한방향으로 밖에 이동을 못한다는 의미인데 일반적인 간선의 경우에는 양방향으로 
이동이 가능하기 때문입니다


## 실수 했던점

### 1. 이 문제는 TC가 여러개 이기 때문에 edges를 계속 초기화를 해줘야 합니다

 edges.clear(); 로 제거를 하지 않는다면, 이전 TC에서 있는 간선이 계속해서 영향을 주기 때문에 꼭 제거를 해줘야 합니다

### 2. 양방향 간선인데 단방향 간선처럼 한 방향만 추가했습니다

  간단한 실수지만, 종종있는 실수이니 꼭 기억해둬야 할 거 같습니다
